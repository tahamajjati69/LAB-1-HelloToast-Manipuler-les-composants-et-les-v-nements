# Hello Toast Android Application

## Description
Hello Toast est une application Android simple développée en Java avec Android Studio.  
Elle permet d'apprendre les bases du développement Android comme l'utilisation des boutons, l'affichage d'un message Toast et la gestion d'un compteur.

L'utilisateur peut :
- afficher un message Toast
- incrémenter un compteur
- voir la valeur du compteur mise à jour à l'écran

---

## Fonctionnalités
- Afficher un message Toast
- Incrémenter un compteur
- Mettre à jour un TextView
- Interaction entre XML et Java

---

## Interface de l'application
L'application contient :

- Bouton TOAST : affiche un message Toast
- Zone centrale : affiche la valeur du compteur
- Bouton Hello Toast : affiche un message
- Bouton COUNT : incrémente le compteur

---

## Technologies utilisées
- Java
- Android Studio
- Android SDK
- XML Layout

## Fonctionnement

### Bouton TOAST
Affiche un message Toast :

Hello Toast!

### Bouton COUNT
Augmente la valeur du compteur de 1 chaque fois que l'utilisateur clique.

### Affichage du compteur
La valeur du compteur est affichée dans un TextView au centre de l'écran.

## Comment exécuter l'application

1. Ouvrir le projet dans Android Studio
2. Attendre la synchronisation Gradle
3. Lancer un émulateur Android ou connecter un téléphone
4. Cliquer sur Run

Cette application permet d'apprendre :
- les layouts Android
- les événements de clic
- l'utilisation de Toast
- la mise à jour de l'interface utilisateur
